# Number Guessing Game

This is a simple python game where a player tries to guess a random number between 0 and 10. 
The game greets the player and also give hints if its too low or high.
 
## How to Run

1. Make sure Python is installed on your computer
2. Open a terminal or command prompt in the project folder.
3. Run the Python file with: 

```bash
Python number_guess.py
```

## Features

* greets players
* Randomly selects a number between 0 and 10
* Lets player guess untill they win
* Gives hints if the guess is too high or too low
* Displays a "You Win!" message when the correct number is guessed
